#### 题目

判断一个数是否是回文数。

> PS: "回文数"是一种数字。如：98789, 这个数字正读是98789,倒读也是98789,正读倒读一样，所以这个数字就是回文数。

#### 示例

输入：`123321`
输出：`true`

输入：`-123321`
输出：`false`

输入：`12321`
输出：`true`

输入：`1`
输出：`true`

输入：`123421`
输出：`false`

#### 分析

既然正读倒读都一样，那么不妨把倒读的数表示出来，看是否与原数相等即可。

另外，回文数都是非负的。

#### 流程

1. 判断是否大于等于零
2. 求出倒读的数
3. 判断是否相等
