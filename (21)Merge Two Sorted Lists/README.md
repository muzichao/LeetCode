#### 题目

合并两个有序链表为一个链表，使新链表依然保持有序。

#### 示例

输入：A = {1, 3, 4, 7, 8, 10};
输入：B = {2, 5, 9, 15};
输出：C = {1, 2, 3, 4, 5, 7, 8, 9, 10, 15};

#### 分析

1. 只需一次查找链表A和链表B的头结点，找到较小结点，然后链接在新链表的尾结点。
2. 如果某个新链表已全部添加，则只需另一个链表剩余的结点接在新链表的尾结点。

#### 流程

1. 判断是否有空链表，有则直接返回另一个链表
2. 合并等长部份
    1. 找到两个链表头结点的较小结点
    2. 找到的结点链接在新链表的尾结点
3. 将不为空的链表直接链接在新链表的尾结点
